# 音速骑士 🏍️

根据手机移动速度自动调节系统媒体音量的安卓应用。

## 功能特性

- 🚀 实时监测手机移动速度（通过加速度传感器）
- 🔊 根据速度动态调整系统媒体音量
- 🎯 三种可自定义配置方案
- 📊 酷炫的动画效果和进度条显示
- 🔄 支持后台持续运行
- ⚙️ 灵活的速度-音量映射配置

## 使用方法

1. 打开应用
2. 选择配置方案（默认"小电驴"）
3. 点击"启动服务"按钮
4. 移动手机，音量会根据移动速度自动调节
5. 点击"停止服务"停止音量控制

## 配置说明

### 默认方案（小电驴）
- 0-10 km/h → 20%
- 10-20 km/h → 30%
- 20-30 km/h → 50%
- 30-35 km/h → 80%
- 35+ km/h → 100%

### 自定义配置
- 点击"编辑方案"进入配置页面
- 支持重命名方案
- 支持添加/删除速度阶梯
- 配置自动保存

## 技术实现

- 使用加速度传感器检测移动
- 低通滤波器去除重力影响
- 平滑算法避免音量突变
- 前台服务 + WakeLock 确保后台运行
- Handler 定时任务实现音量控制

## 权限说明

- 前台服务权限：保持后台运行
- 通知权限：显示前台服务通知
- 音频设置权限：调节系统音量
- WakeLock权限：防止休眠
- 加速度传感器：自动获取，无需额外权限

## 构建说明

1. 使用 Android Studio 打开项目
2. 同步 Gradle 依赖
3. 连接安卓设备或启动模拟器
4. 点击运行按钮构建并安装应用

## 系统要求

- Android 7.0 (API 24) 及以上
- 支持加速度传感器的设备
